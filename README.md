# 🚚 AI Load Consolidation Optimization Engine

This project builds an AI-powered logistics optimization system that intelligently consolidates shipments to improve vehicle utilization and reduce transportation trips.

The system analyzes shipment data and groups shipments into optimal truck loads based on delivery date, destination, and vehicle capacity.

An interactive dashboard built with Streamlit allows users to upload shipment datasets, run optimization, and visualize results such as trips reduction and truck utilization.

## Tech Stack
Python, Streamlit, Pandas, Plotly

## Run the Project
pip install -r requirements.txt  
streamlit run app/dashboard.py
